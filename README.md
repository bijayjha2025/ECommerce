### Wear Nepal

WearNepal is a conceptual e-commerce platform designed to showcase the traditional and modern clothing culture of Nepal. The project imagines a digital marketplace where locally crafted garments and cultural textiles can be discovered, appreciated, and purchased.

The goal of the project is twofold:
1. Celebrate Nepali craftsmanship and culture
2. Strengthen modern frontend development skills using React

This project currently focuses on the frontend architecture and user interface, simulating a real online shopping experience while preparing for future backend integration.

## Live Demo
https://wearnepal-bzvj.vercel.app/ 

## Core Features
- Landing / Hero Section: A visually engaging entry point introducing a featured Nepali fashion collections.
- Product Catalog: Browse a curated list of products inspired by Nepali traditional clothing and materials.
- Search and Filtering: Allows users to quickly find products based on categories or search keywords.
- Shopping Cart : Users can add, remove, and update products inside a persistent cart system.
- Wishlist: Users can save items to a favorites list for later viewing.
- Authentication Interface: A simple Frontend UI for login.

## Project Motivation
Nepal has a deep and diverse textile heritage. Many traditional clothing styles and handmade garments are produced by local artisans but remain underrepresented in modern digital marketplaces.

WearNepal imagines what a dedicated platform for Nepali fashion could look like if built with modern web technologies.

While this version is a frontend prototype, the long-term vision includes transforming it into a fully functional e-commerce system.
